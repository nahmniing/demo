บริการพร้อมส่ง ขนาด M

พร้อมกล่องสีฟ้าขนาด 17 x 26 x 9 ซม.
(เทียบเท่ากับกล่องไปรษณีย์ขาวแดงขนาด ข. สามารถบรรจุสิ่งของได้ไม่เกิน 3 กิโลกรัม)
65 บาท/กล่อง (จำหน่ายแพ็คละ 10 กล่อง)

SKU: PROM-01 Categories: Prompt Post
Description

"บริการพร้อมส่ง (กล่องสีฟ้า) เป็นบริการส่งพัสดุด่วนเทียบเท่ามาตรฐาน EMS ในราคาเหมาจ่ายตามขนาดกล่องที่รวมค่าบริการและค่ากล่องแล้ว (กล่องสีฟ้า) ฝากส่งได้ที่ช่อง Fast Lane ณ ที่ทำการไปรษณีย์ในพื้นที่กรุงเทพฯ และปริมณฑลได้ทันทีไม่ต้องกดบัตรคิว

ขั้นตอนการรับบริการ
เมื่อชำระเงินเรียบร้อยแล้ว ระบบจะส่งข้อมูลการจองบริการพร้อมส่งไปยัง Application Prompt Post ที่ลงทะเบียนไว้
ลูกค้าจะต้องนำ ""รายการจอง"" ที่มีบาร์โคดจาก Application Prompt Post ไปแสดงแก่เจ้าหน้าที่ช่องจำหน่ายกล่องเพื่อรับบริการพร้อมส่ง (กล่องสีฟ้า) ณ ที่ทำการไปรษณีย์ที่ลูกค้าได้เลือกไว้

ขั้นตอนการใช้บริการ Prompt Post
เมื่อลูกค้าแพคสินค้าลงในกล่องสีฟ้าเรียบร้อยแล้ว ให้ทำจ่าหน้าผู้รับล่วงหน้าบนระบบ Prompt Post เท่านั้น โดยเลือกเมนู ""เตรียมการฝากส่ง"" และเมนู ""ทำจ่าหน้า"" เลือก ""บริการพร้อมส่ง"" ระบบจะพิมพ์จ่าหน้าพร้อมแถบบาร์โคดบริการอัตโนมัติ สำหรับติดบนกล่องสีฟ้าได้ทันที ( ทั้งนี้ ไม่สามารถเขียนจ่าหน้าด้วยลายมือได้) นำกล่องสีฟ้าที่ติดจ่าหน้าจากระบบ Prompt Post พร้อม ""ใบส่งมอบสิ่งของ"" ไปส่ง ณ ที่ทำการไปรษณีย์ช่อง ""Fast Lane"" โดยไม่ต้องกดบัตรคิว ระบบจะแจ้งสถานะการนำจ่ายให้อัตโนมัติ (Push Notification) ช่วยให้สะดวกไม่ต้องตรวจสอบสถานะด้วยตนเอง

ข้อมูลของ Application Prompt Post
Prompt Post เป็นเว็ปและแอปพลิเคชันช่วยจัดการส่งสิ่งของทางไปรษณีย์ให้แก่ผู้ขายสินค้า e-Commerce ซึ่งมีจุดเด่นการใช้งาน ดังนี้
- มีระบบการพิมพ์จ่าหน้าพร้อมแถบบาร์โค๊ดอัตโนมัติสำหรับผนึกบนกล่อง/ซอง
- มีระบบพิมพ์ “ใบส่งมอบสิ่งของ” ให้ที่ทำการไปรษณีย์สแกนบาร์โค๊ดเพื่อส่งเข้าระบบครั้งเดียวทุกกล่องได้ทันทีไม่ต้องสแกนทีละกล่อง
- มีระบบแจ้งสถานะการฝากส่งให้ผู้ขายทราบโดยอัตโนมัติ (Push Notification) ไม่ต้องพิมพ์ค้นหาด้วยตนเอง

ลูกค้าที่ต้องการสั่งซื้อบริการพร้อมส่งต้องลงทะเบียนเป็นสมาชิกระบบ Prompt Post ของบริษัท ไปรษณีย์ไทย จำกัด หรือเป็นสมาชิกอยู่แล้วจึงสามารถซื้อบริการพร้อมส่งได้ ลงทะเบียนผ่านเว็ปไซต์ (QR code) ลงทะเบียนผ่านมือถือ (QR code)
